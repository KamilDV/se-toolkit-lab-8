# Available Tools

## LMS Tools (via MCP)
- `lms_health` - Check LMS backend health
- `lms_labs` - List all available labs
- `lms_pass_rates` - Get pass rates for a specific lab
- `lms_scores` - Get scores for a specific lab
- `lms_submissions` - Get submissions for a specific lab

## Usage Notes
- Always call `lms_labs` first if the user asks about labs without specifying which one
- When a lab parameter is needed, ask the user to specify if not provided
